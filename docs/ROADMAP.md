# 실행 로드맵 — 지속형 왕국 (KoA-like 솔로)

> 방향·원칙은 [기획서 §13](기획서.md). 이 문서는 **실행 단위**로 쪼갠 작업 목록(하나씩 구현→검증→커밋).
> 작업 방식: 단일 소스 원칙 유지 · 각 단계 `node build.js && node verify.js`(+ 필요시 sim/브라우저) 통과 후 커밋 · game.js/engine.js 순수 유지(실시각·DOM은 ui 레이어).

범례: `[ ]` 예정 · `[~]` 진행 중 · `[x]` 완료

---

## Milestone A — 지속형 성장 루프 (기반) ← **지금 여기**

"매치 → 지속 왕국" 전환의 토대. 나머지(B·C)가 전부 이 위에 얹힘.

### A1. 국력(Might) 점수 — 성장 척추 ← **첫 걸음**
왕국의 강함을 단일 수치로 요약, 상단 상시 표시. 모든 성장이 이 숫자를 올린다(KoA의 중심 감각).
- [ ] `game.js computeMight(g, side="P")` — 가중합: 성 레벨 · 경제/생산 건물 레벨합 · 완료 연구 수 · 병력(주둔+부대, 티어 가중) · 성벽 · 영웅(등급합). 상수는 초기값 후 튜닝.
- [ ] API export + `game.js` 스모크(각 요소 증가 시 monotonic 상승).
- [ ] `ui.js`: 상단 자원바에 `⚔ 국력 N` 배지. 증가 시 가벼운 피드백(색/애니).
- [ ] (선택) 적 세력 국력도 계산해 "적 대비" 표시 → 위협 체감.
- **검증**: newGame baseline; 건물/연구/병력/영웅 늘리면 증가. 헤드리스 스모크.
- **크기**: 소(0.5일). 의존성 없음.

### A2. 마일스톤 사다리 — 장기 목표·해금
국력/시간 문턱마다 보상·해금. "다음 목표"가 항상 존재.
- [ ] `game.js MILESTONES` = [{id,name,need:{might?,turn?},reward,unlock,desc}]. 예: 국력100→자원+"T2 해금", 250→"드래곤 알", 500→"3티어·다수세력", …
- [ ] `milestoneTick(g)` (endTurn 내) 충족분 완료·보상·해금 플래그. 저장에 진행 포함.
- [ ] **해금 게이팅 연결**: 특정 티어/건물/유닛이 마일스톤 unlock에 걸리게(생산·건설 함수에서 체크).
- [ ] `ui.js`: 목표 패널에 현재 마일스톤 진행바(국력 X/Y). 온보딩 퀘스트(§12)와 공존 또는 통합.
- **검증**: 국력 올리면 마일스톤 순차 완료·보상·해금 반영(스모크).
- **크기**: 중(1일). A1 의존.

### A3. 비종결 플레이 — 지속 왕국 전환
적 함락·레이드가 "게임오버"가 아니라 진행 이벤트. 왕국은 계속.
- [ ] `game.js checkVictory` 재설계: `over` 기본 끔. 적 성 함락 = 영토/보상 + (옵션) 새 세력 리스폰. "정복 완료"는 마일스톤으로.
- [ ] 패배 처리 재설계: 내 성 함락 = 즉시 종료 대신 큰 손실·재건(또는 난이도 옵션). 착수 전 세부 재확인.
- [ ] `ui.js endGame` 흐름: 승리 모달 → "마일스톤 달성" 토스트.
- **검증**: 적 함락해도 계속 진행. sim은 "정복 횟수" 지표로 전환.
- **크기**: 중(1일). 승패 구조 변경 → **착수 전 사용자 재확인**.

### A4. 오프라인 누적 — "돌아오면 자라있음"
자리 비운 실시간만큼 자원·생산·연구·건설이 상한 내 진행(결제벽 아닌 재미 훅).
- [ ] 저장에 `savedAt`(실시각 ms) 기록 — `ui.js` saveSnapshot/saveLocal (`Date.now()`, **UI 레이어 한정**).
- [ ] 로드 시 정산: 경과 틱 = (now−savedAt)/틱길이, 상한 clamp(예: 최대 8~24시간치). `offlineTick(g,ticks)`로 자원·생산·연구·건설 일괄 진행(전투·AI는 요약/스킵).
- [ ] "자리 비운 N시간 동안 …" 요약 토스트/모달.
- **검증**: savedAt 조작으로 경과 시뮬 → 자원/생산 증가·상한 동작. game.js 순수 유지(sim/verify 무영향).
- **크기**: 중(1일). A1~A2 위에서 의미. **실시각은 ui.js에서만**.

## Milestone B — 위협 & 세계 (escalating + 탐험)

성장 루프 위에 "다음 위험"과 탐험. (A 완료 후 상세화)

### B1. 다수 세력
- [ ] 적 2~3세력(§2 약속). newGame에 세력별 성·본대·AI 상태. aiTurn 세력별 루프.
- [ ] 세력별 개성(초기 병종 편향·공격성) — 영웅 라인업으로 차별(§4-B).

### B2. 시즌형 침공 (escalating)
- [ ] 시간·국력에 비례해 웨이브·보스 침공 강화. "시즌" 스케줄(N턴마다 대규모 침공 예고→도래).
- [ ] 예고 UI + 대비 시간 → 긴장·페이싱.

### B3. 안개·탐험·영토 확장
- [ ] fog of war ON(§10에서 껐던 것 복원). 정찰/시야로 맵 개방.
- [ ] 노드/타일 점령 = 영토. 영토가 국력·수입에 기여.

### B4. 스케일링 PvE
- [ ] 둥지·보스가 국력/시간에 비례해 강해짐(항상 도전거리).
- [ ] 월드 보스 이벤트(고보상·고난도).

## Milestone C — 정체성 (드래곤·영웅 깊이)

작동하는 왕국 위에 얹는 훅. (B 진행 중/후 상세화)

### C1. 드래곤 — KoA 상징, 성장 축
- [ ] 알 획득(마일스톤/레이드) → 부화 → 육성(자원/경험치) → 전투 참여(강력 유닛/광역 버프) → 스킬.
- [ ] 드래곤 성장이 별도 진행 축 = 장기 동기.

### C2. 영웅 깊이
- [ ] 스킬(트리) · 장비(획득·장착) · 레벨링 — 현 등급제 위에.

### C3. 병종·티어 확장
- [ ] 티어 확장(3→5+), 공성병(성벽 특효), 병종 다양화.
